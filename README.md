# leetcode-
自己制作的一个leetcode客户端，主要用于学习。

## 依赖
本项目基于react-native，请自行安装如下命令：

`node` `npm` `rnpm` `react-native` `adb` `java` 

## 项目启动方式

```$xslt
//请先连接安卓虚拟机

git@github.com:DarkblueTang/leetcode-android.git

cd leetcode-android.git

react-native run-android
```

## 项目打包方式
```$xslt
cd android

gradle assemble
```
